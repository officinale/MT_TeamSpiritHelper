# Readme
MT_TeamSpiritHelperは就業管理システムであるTeamSpiritの入力支援スクリプトです。
なお、スクリプトの仕様上、動作する環境、条件に制限があります。

本スクリプトはAutoHotKey V2を使用しています。
本スクリプト作成時に動作確認を行っているバージョンはV2.0.11です。

### 制限事項
##### 処理全体の制限
* 本スクリプトはブラウザの表示倍率を100%としていることを前提としています。スクリプトの動作前には表示倍率を100%へ戻すようにしてください。
* ブラウザの入力支援機能等が有効となっている場合、スクリプトの動作が阻害されることがあります。その場合はブラウザの設定を変更してください。

##### 申請処理の制限
* 現時点では申請処理の自動入力対象はflex勤務のみです。その他の場合は誤動作する恐れがあるため、あらかじめ申請処理を手動で済ませてください。
* 申請の項目が「勤務」以外となる場合は手動で申請を行ってください。
##### 出退勤入力の制限
* 既に出退勤情報に値が入力されている場合、処理はスキップされます。
* 現時点では時刻の入力、保存処理を自動で行いますが、申請処理は行いません。申請処理はスクリプトの自動入力後に手動で行う必要があります。
##### 工数入力の制限
* 既に工数に値が入力されている場合、処理はスキップされます。
* このスクリプトでは工数入力を自動で行いますが、スクリプトの仕様上客先勤務をジョブの先頭に配置しておく必要があります。
* 客先勤務のジョブを複数入力する処理には対応していません。
* 製品分野等の入力時にはお気に入りにある項目の先頭を選択する仕様になっています。お気に入りの項目を切り替えるような動作には対応していません。
* このスクリプトで自動入力されるのは以下の項目です。これ以外の入力が必要な場合はスクリプト処理の後で手動入力を行ってください。
    ```
    製品分野、業務区分、技術要素1～3、知識、技能、アウトプット名称
    ```
* 労懇などで客先勤務以外の工数が発生した場合はTimesheet.xlsxの値が想定する値からズレる恐れがあるため、工数が正しい値になっているか確認するようにしてください。
##### 勤務場所、業務内容、備考の制限
* 勤務場所、業務内容、備考の情報は、記載状況に関わらず上書きする仕様となっています。既に記載された情報を残す場合は、スクリプトが処理をする日付から外す必要があります。

### 事前準備
##### 1.AutoHotKey V2の入手
以下のURLからAutoHotKey V2をダウンロードし、ahkファイルを実行出来る状態にする必要があります。
https://www.autohotkey.com

### 使用方法
##### 1.就業管理情報の入力
Timesheet.xlsxを開き、SAMPLEシートをそのまま利用するか、SAMPLEシートをコピーして任意のシートを作成します。
シートを作成した場合は該当シートが一番左に来るようにタブの位置を移動してください。

その後、SAMPLEシートに従い就業管理情報を入力し、保存します。

##### 2.tsvファイルの生成
tsvファイルを生成するために、ExportSheetToTsv.vbsへTimesheet.xlsxをD＆Dします。
vbsの設定はデフォルトでは一番左のシートのみを対象としてファイル生成を行います。
処理が完了すると、Timesheet_[シート名].tsv というファイルが生成されます。

##### 3.スクリプトの実行
MT_TeamSpiritHelper.ahkを実行します。正常に起動するとウインドウが表示されます。
表示されたウインドウに 2. で生成したtsvファイルをD＆Dします。
tsvファイルのロードに成功すると「Employment info has been loaded.」というメッセージが表示されます。

ウインドウ上には開始する日付と終了する日付を入力する欄があります。スクリプトが処理をする日付はその範囲内で行われます。入力したい日付に応じて適時変更してください。

その後、ブラウザのTeamSpiritのページで勤務表のタブを表示し、F1キーを押下すると自動入力処理を行います。

動作を途中で停止したくなった場合は、ウインドウにAbortの表示が出るまで「Pause（Break）」キーを押し続けてください。
また、「Ctrl + Esc」キーを押下すると、アプリを強制的に終了することが出来ます。
